# Role-Based Authentication System

A production-ready MERN stack template with Access Token and Refresh Token authentication.

## Roles
- **Admin**: Hardcoded authentication, no signup.
- **Student**: Self-signup and login.
- **Instructor**: Self-signup and login.

## Structure
- `/backend`: Node.js + Express API.
- `/frontend`: React + Vite + Tailwind CSS UI.

## Features
- JWT Access and Refresh Tokens.
- Role-based Protected Routes.
- MongoDB with Mongoose Schemas.
- OTP and Email utilities.
